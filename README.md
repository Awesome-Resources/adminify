# Adminify
- An Admin dashboard based on [Vuetify](https://vuetifyjs.com/) .
- API is built on [AdonisJs](http://www.adonisjs.com/) . 
- Also welcome to [Adonis China](https://adonis-china.org/) .

## Getting start
``` bash
npm install
npm run dev
```
> Use `cnpm` instead `npm` in China
